# Reference Mockup

This directory contains the visual specification for the Investment Dashboard.

## Files

- `Investment_Dashboard.html`: The canonical visual reference. It defines the dark theme (using `oklch` colors), the layout, typography, and component styling.

## Usage

- **Do not modify** these files.
- **Do not run** this HTML as part of the application.
- Use it to extract color tokens, spacing, and CSS rules for the Streamlit implementation in `app/ui/`.
- If the Streamlit implementation drifts from the design, refer back to this HTML to correct it.

The Streamlit implementation lives in `app/ui/`.
